FR1EV SECURITY - Privacy Policy

Last Updated: 2026

FR1EV SECURITY is designed to moderate and protect Discord servers. This Privacy Policy explains what data may be processed and how it is used.

Data We Process

  • Discord user IDs
  • Server, channel, role, and message IDs
  • Message content when required for moderation, logging, commands, TTS, or automod
  • Server configuration such as roles, logs, tickets, verification, levels, and dashboard settings

How Data Is Used

  • Moderation actions like kick, ban, timeout, warns, and role changes
  • Logging and server management
  • Verification, tickets, leveling, automod, and AI features
  • Dashboard permission checks and server configuration

Data Storage

  • Data may be stored temporarily or persistently for bot features
  • No passwords are collected
  • Data is not sold

Data Sharing

FR1EV SECURITY does not sell, share, or distribute user data to third parties except where Discord, Roblox, or configured service APIs are required for a feature to work.

Data Removal

Server owners can remove server data by disabling features, removing the bot, or contacting the developer.

Contact

For privacy questions, contact the bot owner through the support server.
