🛡️ FR1EV SECURITY – Privacy Policy

Last Updated: 2026

FR1EV SECURITY (“the bot”) is designed to moderate and protect Discord servers. This Privacy Policy explains what data may be processed and how it is used.

📊 Data We Process

  • Discord User IDs
  • Server (Guild) IDs
  • Channel IDs
  • Message content (only when required for commands, logging, or moderation)
  • Server configuration (roles, logs, settings)

⚙️ How Data Is Used

  • Moderation actions (kick, ban, timeout, warnings)
  • Logging system (messages, joins, actions)
  • Bot commands & features
  • Improving performance and stability

🔒 Data Storage

  • Data may be stored temporarily or persistently
  • No sensitive personal data (passwords, emails) is collected
  • Data is never sold

🚫 Data Sharing

FR1EV SECURITY does not sell, share, or distribute user data to third parties.

🧹 Data Removal

Server owners can remove data by removing the bot or contacting the developer.

🔐 Security

We take reasonable steps to protect data and prevent unauthorized access.

📌 Changes

This policy may be updated. Continued use of the bot means you accept changes.

📩 Contact

For questions, contact the bot owner.
