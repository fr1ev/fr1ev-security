What you can control

The dashboard is made for server owners and admins who want a simple way to manage protection, moderation, and setup without running every command manually.

Automod Settings

Configure spam filters, invite blocking, link blocking, mention limits, bad word filters, timeout actions, and server safety rules.

Moderation Tools

Manage bans, kicks, timeouts, untimeouts, warnings, role controls, and permission-safe actions from the dashboard.

Logging

Set log channels for moderation actions, joins, leaves, bans, deleted messages, edited messages, tickets, and important server events.

Verification

Set up Discord verification and Roblox linking so members can connect their Roblox account and get verified safely.

Leveling

Control leveling, XP rewards, custom rank cards, level roles, leaderboards, vote rewards, and daily activity systems.

Tickets & Support

Manage support ticket settings, staff roles, channels, panels, and server support tools from one place.

How to use the dashboard

Follow these steps to connect your server and start managing FR1EV SECURITY.

Add FR1EV SECURITY to your server

Invite the bot with the correct permissions so it can manage moderation, automod, logging, verification, and setup features.

Open the official dashboard

Go to dashboard.fr1ev.xyz and log in with your Discord account.

Select your Discord server

Pick a server where you have admin permissions and where FR1EV SECURITY is already installed.

Configure protection features

Enable automod, logging, verification, leveling, tickets, welcome messages, and other server protection tools.

Save and test your setup

Use the dashboard and bot commands to confirm that your server settings work correctly.

If your server does not appear in the dashboard, make sure you are logged into the right Discord account and that the bot is already in the server.

Open the dashboard now

The official dashboard is hosted at dashboard.fr1ev.xyz and connects directly to FR1EV SECURITY settings.
